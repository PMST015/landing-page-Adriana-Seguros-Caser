# Landing Page - Adriana Seguros Caser 🛡️

Este proyecto consiste en el diseño y desarrollo de una Landing Page profesional para una agente certificada de **Seguros Caser**. El objetivo principal es la captación de clientes potenciales (Lead Generation).

### 🎯 Objetivo del Proyecto:
Crear un embudo de conversión directo donde los usuarios interesados en seguros puedan validar la certificación de la agente y contactarla de inmediato vía WhatsApp para una asesoría personalizada.

### 🛠️ Tecnologías usadas:
* **WordPress / Elementor**
* **Optimización de Conversión (CRO):** Ubicación estratégica de botones de llamado a la acción (CTA).
* **Integración con WhatsApp Business:** Automatización del flujo de contacto.

### 📈 Resultados:
* Interfaz que transmite confianza y profesionalismo (autoridad de marca).
* Canal directo de comunicación que facilita la conversión de visitantes en clientes reales.
